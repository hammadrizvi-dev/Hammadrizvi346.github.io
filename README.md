## 🛠️ Technical Arsenal

This portfolio isn't just a static page; it's a reflection of my core competencies:

* **Languages:** Java (DSA Focus), Python (Data Science Focus), C, JavaScript (ES6+).
* **Web Tech:** HTML5, CSS3, Bootstrap 5, Glassmorphism UI.
* **Data Science:** Pandas, NumPy, Matplotlib.

---

## ✨ Key Features

-   **Dynamic Project Rendering:** Projects are loaded dynamically from a `data/projects.json` file using JavaScript, making it scalable and easy to maintain.
-   **Premium Dark Tech Theme:** A professional UI built with a deep navy palette and neon cyan accents.
-   **Interactive Skill Dashboard:** Glowing progress bars and hover-scale tool grids.
-   **Bento Box Layout:** Modern grid system inspired by top-tier SaaS products.
-   **Smooth Animations:** Integrated with AOS (Animate On Scroll) for a fluid user experience.
-   **Encrypted Messaging Focus:** Highlighting projects like **Chatify** featuring E2EE security.
